# DDD-Study

DDD是一种架构思想，不是一种具体的框架，所以强调的是**思想设计**，而非具体实现，也就是DDD不像Spring框架那样有标准的代码框架和结构，所以
DDD的实现方式有很多种，这里只是记录自己在学习过程中的一些心得和笔记。
